# lev01 Penetration Testing Tool

## 
أداة اختبار اختراق متعددة الاستخدامات مكتوبة بلغة Python، تحتوي على ميزات لفحص المنافذ، جمع المعلومات، والتقاط الحزم.

## كيفية الاستخدام
1. فحص المنافذ: 
    ```sh
    python main.py
    ```
2. جمع المعلومات:
    ```sh
    python main.py
    ```
3. التقاط الحزم:
    ```sh
    python main.py
    ```
